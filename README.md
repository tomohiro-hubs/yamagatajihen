# 山形事変 (Yamagata Jihen)

山形県文翔館で開催される歴史的建築と名車の展示イベント「山形事変」の公式ランディングページです。

## プロジェクト概要

大正ロマンを感じさせる重要文化財「文翔館」と、往年の名車（クラシックカー）が共演するイベントの告知ページです。
高級感、歴史、重厚さをテーマにデザインされています。

## 特徴

*   **レスポンシブデザイン**: Tailwind CSSを使用し、PC、タブレット、スマートフォンに対応。
*   **フォント**: Google Fontsより「Noto Serif JP（明朝体）」と「Noto Sans JP（ゴシック体）」を採用し、洗練された印象を演出。
*   **アニメーション**: CSS Animationを使用したフェードイン演出。
*   **インタラクション**: ヘッダーの追従、スムーススクロールなどのJavaScript実装。

## ファイル構成

*   `index.html`: メインページ（HTML, CSS, JSを含むシングルファイル構成）

## 開発・デプロイ

このページは静的なHTML/CSS/JSのみで構成されており、特別なビルドプロセスは不要です。
画像ファイルが見つからない場合は、自動的にUnsplashの画像にフォールバックするよう設定されています。
